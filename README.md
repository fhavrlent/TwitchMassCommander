# Twich Bulk Commander

1. Install [Node.js](https://nodejs.org/en/)
2. Install yarn

```bash
npm i yarn -g
```

3. [Generate Oauth token](https://twitchapps.com/tmi/)
4. Copy `example.env`, rename to `.env` and fill it out.
5. Start the script

```
yarn start
```

Gist should containt one command per line.
